# UpLoL

## About

> A MERN stack, mobile-first application for sharing funny, tasteful images for others to enjoy. Reliant on JWT for user sessions, bcrypt for password hashing, Cloudinary for storing and hosting-of images due to database limitations, and MongoDB & Mongoose with Apollo GraphQL for queries and mutations to the database. Logged in users can create a profile, follow users, and like & comment on posts. The Discover page and user search allows users not logged in to still have a fun experience without being required to log in.

## Usage

> No installation required, just launch application at https://uplol.herokuapp.com/ from any computer or smart device.
